OpenGraphMagic is a WordPress plugin that automatically generates images for Open Graph tags using external services like Pikwy and ScreenshotOne.

== Description ==

OpenGraphMagic enhances your website's social media integration by automatically generating and embedding Open Graph images tailored for social platforms. This ensures that your content looks appealing when shared on social media.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/opengraphmagic` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->OpenGraphMagic screen to configure the plugin.

== Localization ==
OpenGraphMagic is fully translation-ready, allowing for community-contributed translations using the unique text domain: 'opengraphmagic'. We encourage contributions to our translation efforts to make OpenGraphMagic accessible to everyone.

== Frequently Asked Questions ==

= What is Open Graph? =

Open Graph is a protocol that enables any web page to become a rich object in a social graph. For example, it is used on Facebook to allow any web page to have the same functionality as any other object on Facebook.

= How does OpenGraphMagic work? =

OpenGraphMagic uses third-party services to generate images based on the content of your website. These images are then used as Open Graph images for better visual representation on social media platforms.
